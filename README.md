# dotfiles

** Designed to be used on Arch linux

## Assumptions
Tela icon pack installed
Breeze theme pack installed


## Usage

1. clone the repository
2. cd into dotfiles
3. run `./install.sh`

## Dependencies
- hyprland
- waybar
- wayland
- gtk3
- qt5
- dunst
- swww
- wlogout
